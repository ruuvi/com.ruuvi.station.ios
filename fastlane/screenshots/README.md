# App Store Screenshots

This is the live folder used by the `Sync App Store Metadata` workflow.

Add one folder per App Store Connect locale:

```text
fastlane/screenshots/en-US/01_home.png
fastlane/screenshots/en-US/02_history.png
fastlane/screenshots/fi/01_home.png
```

Add real `.png` or `.jpg` screenshots directly under the matching locale folder before running the
workflow with screenshot sync enabled.
